# Blocky



## Description
Me trying to make a voxel in OpenGL 3.3+

## Libraries
- GLFW
- GLM
- GLEW

## Build
Install cmake 3.16+ and run
```bash
mkdir build
cd build
cmake ..
```

To run, simpily run (in build directory)
```bash
./blocky
```
